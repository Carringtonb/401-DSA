
## Balanced Parens
*Author: Carrington Beard*

---

### Problem Domain
Create a method to check a string argument for a balanced amount of opening/closing parenthesis.
---

### Inputs and Expected Outputs

| Input | Expected Output |
| :----------- | :----------- |
| [](({)}) | True |
| ](){} | False |


---

### Big O


| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |
