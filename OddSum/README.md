
## Odd Sum of Binary Search Tree
*Author: Carington Beard*

---

### Problem Domain
Find the sum of all the odd numbers in a binary search tree.

---

### Inputs and Expected Outputs

| Input | Expected Output |
| :----------- | :----------- |
| [1] | [19]|
| [2] [5]|
| [4] [6] [13] [8]|


---

### Big O


| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |
